Implementation of AI powered Autonomous system system for electric solar vehicle.
Level of Automation: Level 2 (partial Automation)

Introduction to vehicle
Infiniti 8.0 an SEV, powered with solar, electronics is a strong and durable vehicle. Manufactured from scratch.
Performed various mechanical, electrical and autonomous testings. Vehicle controlled with a BLCD motor and transmission, providing sustainability.
Automation in vehicle implemented with autonomous levels partial driving, where vehicle helps driver for cruise controlling and navigate through lanes.
Machine learning implementation with cone detection and mapping the route with dynamic control.

Vehicle Automation features include self navigation, object detetion and speed control, make vehicle take auto decisions to travel a path.

Vehicular Automation stack SOFTWARE: Object datection, Lane assistance, Self navigation, DAQ (Data aquisition) and analysis, Computer vision

HARDWARE: Raspberry pi 4b, Linear Electric Actuator, Motor driver L298N, web cam, 12V DC accumulator
